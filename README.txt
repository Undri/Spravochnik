Справочник представляет собой массив словарей. Каждый словарь - отдельная запись
о человеке. Соответственно, обращения к любому свойству человека происходят
по ключам (first_name, last_name, phone_number, birth_date).


Основные функции:

    main() представляет собой гланое "меню" программы. Цикл, в который всегда
    возвращаются другие функции-действия (add, show, find, remove)

    finder() обеспечивает поиск по предлагаемым параметрам (только имя,
    только фамилия, полное имя, телефонный номер)

    adder() позволяет занести человека в справочник. Если по введенному
    имени и фамилии было определено, что такой человек уже занесен в
    справочник, функция дает возможность изменить информацию найденного
    человека или группы лиц, вызывая функцию changer()

    changer() позволяет вносить изменения в запись о человеке. Если была
    вызвана из finder(), который вывел список лиц, дает возможность выбрать
    из списка человека, информацию о котором нужно изменить.

    remover() удаляет записи о человеке из справочника. Требует указать имя
    и фамилию.


Вспомогательные функции:

    printer() печатает содержимое справочника. Использует библиотеку
    PrettyTable (pip3 install PTable)

    full_name_checker() необходима для установления, занесен ли человек в
    справочник. Возвращает 0, 0 если человек не занесен. Возвращает -1, i
    если человек занесен, где i - индекс человека в справочнике.

    sort_spravochnik() сортирует справочник.

    phone_adder() функция для корректной обработки номера телефона перед
    занесением в справочник.

    birth_adder() функция для корректной обработки даты рождения перед
    занесением в справочник. Использует библиотеку datetime.
